Описание
-----------
Менеджер для работы с Cryptopro

Установка пакетом
-----------
Для локальной разработки::
    pip install -e packages/cryptopro_manager

Импорт
-----------
Проверка::
    from managers.cryptopro_manager import CryptoproManager


Удаление
-----------
Удалить пакет::
    pip uninstall cryptropro_manager

Для создания пакета
https://docs.python.org/3.10/distutils/introduction.html#distutils-simple-example
https://docs.python.org/3.10/distutils/sourcedist.html
::
    python setup.py sdist
